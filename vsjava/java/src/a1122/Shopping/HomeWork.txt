getter와 setter 메소드를 통해 값을 조회하고 변경하도록 하세요.
생성자를 이용하여 학생 정보를 초기화하고, 평균 점수를 계산하는 메소드를 추가하세요.

// 학생 객체 생성
Student student1 = new Student("홍길동", 20231234, 90, 85, 92);

학생 정보 출력
